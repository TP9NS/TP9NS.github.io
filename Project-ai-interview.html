프로젝트 개요

6인 팀 · 2025.07–2025.09

AI를 활용해 사용자 맞춤 면접 질문을 생성하고, 표정/동작 분석과 타임스탬프 기반 피드백을 제공하는 플랫폼입니다. React Native 앱과 LangGraph 기반 분석 파이프라인을 연동했습니다.

Spring BootReactReact NativeLangGraphAWS

주요 기능

  • 사용자 맞춤 면접 질문 생성
  • Koelectra 기반 꼬리질문 판단 및 흐름 제어
  • 표정/동작 분석 채점 + 타임스탬프 피드백
  • 결과 저장 및 관리자 통계
  • 모바일 앱 연동, 소셜 로그인

기여 내용

  • 면접(답변) 흐름 설계 및 구현
  • LangGraph 파이프라인/임베딩 개선
  • AWS ECR/ECS, IaC 배포 파이프라인
  • OAuth2 소셜 로그인, JWT 인증/인가

기술 스택

Backend

  • Spring Boot, MariaDB, Redis
  • Flask/FastAPI (분석 API)

AI

  • LangGraph, MediaPipe, Deepface, llama3
  • ChromaDB

Frontend/Mobile

  • React, Next.js, React Native, Tailwind

Infra/CI

  • GitHub Actions, AWS ECR/ECS, IAM
  • Terraform, CloudWatch, Nginx

트러블 슈팅

문제 1

영상/음성 동시 전송으로 처리 지연

해결: 필요한 단계만 동기 처리, 나머지는 비동기 처리 + Redis 버퍼.

문제 2

분석 종료 전 결과 요청으로 동기화 붕괴

해결: 웹소켓 유지 + Redis Done 플래그로 완료 기준 통일.

문제 3

수평 확장 시 웹소켓 서버 불일치로 이벤트 유실

해결: Redis Pub/Sub으로 분배.

문제 4

모바일 OAuth2 리다이렉션 정책으로 로그인 실패

해결: 브릿지 컨트롤러로 모바일/웹 세션 분리, 토큰 교환 흐름.
